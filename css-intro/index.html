CSS coding styles

Browser Defaults you have not supplied any css code
Inline this is for a single specific element (tag) instance. the element contains a parameter called style="". style="rule; [rule; ...]" style="color:red; width:50px;"
Embedded coded in your head area of the web page. is started with the style element and contains the selectors and rules for the desired css. applies ONLY to the physical page where the code is placed. Affects all selectors on the page <style>
selector {rule; [rule; ...]}
[selector {rule; [rule; ...]} ...]
</style>		 <style>
p {font-size:large; font-weight:bold;}
blockquote {width:350px;}
</style>
Stylesheet similar to embedded however, the selectors and rules are code in a separater phyiscal file. this file can be shared throughout the entire web site. thus, all css is in one common place. selector {rule; [rule; ...]}
[selector {rule; [rule; ...]} ...]		 p {font-size:large; font-weight:bold;}
